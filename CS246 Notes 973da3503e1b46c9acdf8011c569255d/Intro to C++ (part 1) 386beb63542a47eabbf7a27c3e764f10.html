Standard Input & Output & Error Stream

<iostream>: access to 3 stream variables:

  1. Standard input: std::cin Type: std::istream (i stands for input) Input operator: >> Note: By default, std::cin ignores whitespace example: 
    # include <iostream>
int main() {
	int x;
	char c;
	std::cin >> x >> c;
	// A integer will be read from input and will be stored in x,
	// then a character will be read from input and will be stored in c.
  1. Standard output: std::cout Type: std::ostream (o stands for output) Output operator: << example:
    #include <iostream>
int main() {
	int x = 5;
	char c = 'a';
	std::cout << x << c << endl;
	// Value of x is sent to output stream, then the value of c is
	// sent to the output stream. Then a newline is sent and the cashe
	// is cleared.
  1. Standard error: std::cerr

Using namespace std

std stands for standard. After calling, objects, variables, functions in standard library can be directly used.

Purpose of using namespace std:

  1. objects/variables/functions in standard library now become global scope
  1. can be directly used
  1. omit the prefix std::

Example:

#include <iostream>
using namespace std;
int main() {
	int x;
	cin >> x;
}
// Now we can use cin directly without including the prefix std::

Advantages & Disadvantages of using namespace:

  1. save time such as less typing
  1. but considered as bad behavior (but ok for this course)
  1. might lead to name collisions

Example:

#include <iostream>
using namespace std;
extern istream cin;
int main() {
	int x;
	cin >> x; // Error occurs: compiler doese not know which cin to use
}

How to detect whether input was successful

Method 1:

std::cin.eof(): If it’s end of file, then this function returns true, otherwise, false.

Method 2:

std::cin.fail(): If it failed to read in input or fail to store the input into variable, then this function returns true, otherwise, false.

Method 3:

std::cin >> x: This command’s return value is a Boolean. If the input & storage is successful, then it returns true, otherwise, false.

Exercise: Write a C++ program to read in integers from input (ignore other inputs), and print all integers read successfully separated by whitespace. (You program should be able to handle invalid input (not integer))

Solution:

#include <iostream>

int main () {
	int x;
	while (true) {
		if (std::cin >> x) {
			std::cout << x << " ";
		} else { // if read fails
			// if reaching the end of file, then break
			if (std::cin.eof()) {
				break;
			} else { // if it's an invalid input (not integer)
				// clear error flag
				std::cin.clear();
				// once an invalid input is read, the error flag is on
				// and will not be turned off automatically
				std::cin.ignore(); // discard the char will be read
			}
		}
	}
}

Additional Comments:

  1. In this course we won’t be dealing with invalid inputs (we may assume that all inputs are valid while doing assignments)
  1. As soon as a library is introduced, we may use any functions in that library (even though it’s not taught in class).

String in CS136: (C)

  1. library #include <string.h>
  1. string literals: using double quotes, cannot be mutated once created

Strings in CS246: (C++)

  • string in C++ resizes automatically

  • string length: x.length()
  • string comparison: ==
  • substring: destination = src.substr(start_index, length);

Exercise:

Write a program that reads from an existing file called “in.txt” and copies the contents of even line numbers to another existing file “out.txt”.

Solution:

Step1:

  • create a new file for C++ program with file extension .cc or .cpp (we use .cc in this course)

    vim copy.cc

Step 2:

  • Include needed libraries: <iostream>, <fstream>, <string>

Step 3:

  • error checking
  • if both files exist, we proceed
  • if any one of the two files does not exist, print a message to standard error (std::cerr) and exit with status code 1
  • check if the files are opened successfully by using function .is_open()

Step 4:

  • start to read from in.txt and output to out.txt

Step 5:

  • first compile program:
    • Linux command: g++ -std=c++14 (Too complicated, we can simplify this process by adding an alias:)
    • alias g++14="g++ -std=c++14 -Wall -g"
  • then run the program:
    • Method 1:
      • Linux command: g++14 copy.cc
      • If compiled successfully, we will see a new executable named “a.out” under the current directory
      • Execute the program: ./a.out
    • Method 2:
      • Linux command: g++14 copy.cc -o copy
      • “a.out” is the default name for compiled program, we can customize the name of executable by running the above Linux command
      • We will see an executable file named “copy” under the current directory
      • Execute the program ./copy
  • In this course, we need to compile and run manually.

Step 6:

  • Test to check if program performs correctly: cat out.txt

The completed program:

#include <iostream>
#include <fstream>
#include <string>

int main() {
	std::ifstream in{"in.txt"};
	std::ofstream out{"out.txt"};
	
	// check if both files exist
	if (!in.is_open() || !out.is_open()) {
		//print an error message to standard error
		std::cerr << "file missing!" << std::endl;
		return 1;
	}
	std::string line;
	int count = 0;
	while (std::getline(in, line)) {
		// update line number
		count += 1;

		// if even line number, then output to file
		if (count % 2 == 0) {
			out << line << std::endl;
		}
	}
}

Input / Output manipulators

  • library: #include <iomanip>
  • when printing integers, C++ prints integers in decimal format by default
  • we can change it to other formats, such as hexadecimal
  • once changed format, we need to change it back to decimal format manually
  • (barely used in this course)

Example:

#include <iostream>
#include <iomanip>

int main() {
	int x = 16;
	// print x in decimal format
	std::cout << x << std::endl;
	// change decimal to hexadecimal
	std::cout << std::hex;
	// now print x in hexadecimal format
	std::cout << x << endl;
	// change back to decimal format manually
	std::cout << std::dec;
	// now it will print x in decimal format
	std::cout << x << std::endl;
}

C++ with Command Line Arguments

  • Function main usually takes no parameters
  • When C++ program interacts with command line arguments, arguments will be taken as parameters of main

Example:

#include <iostream>

int main(int argc, char* argv[]) { // identifier names can be different
	std::cout << "Number of arguments: " << argc << std::endl;
	for (int i = 0; i < argc; i++) {
		std::cout << argv[i] << std::endl;
	}
}
// Note: the first item in argv is always the name of the program
// That's why in most cases, we start the iteration from i = 1
